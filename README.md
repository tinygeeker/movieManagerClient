# 🎬 电影管理器

一个跨平台的电影管理客户端，支持Windows和macOS系统，让你的电影收藏井然有序！✨

## 📋 功能特性

- 🎯 **智能扫描**：自动扫描并管理指定文件夹下的所有电影
- 📊 **详细信息**：显示电影的详细信息，包括格式、时长、文件大小、分辨率等
- 📁 **文件夹导航**：默认选择同级目录开始检索，可轻松切换上下级目录
- 🎨 **卡片式显示**：电影卡片式展示，可切换显示数量，每页最多显示100部电影
- 👀 **悬浮详情**：鼠标悬浮时显示电影详情明细，一目了然
- ▶️ **一键播放**：点击电影卡片可直接播放电影，无需繁琐操作
- 🖼️ **美观封面**：支持从视频中提取帧作为封面，自动生成彩色预览
- 🇨🇳 **中文界面**：全中文界面，操作简单直观，适合所有用户

## 🛠️ 前提条件

### 基本运行条件
- 🟢 Node.js
- 📦 npm 或 yarn
- ⚡ Electron

### 可选依赖（用于获取更多视频信息）
- **ffprobe**：用于提取视频元数据（时长、分辨率、编码等）
- **ffmpeg**：用于从视频中提取帧作为封面

#### 安装方法
1. **Windows**：
   - 下载并安装 [FFmpeg](https://ffmpeg.org/download.html#build-windows)
   - 将ffmpeg添加到系统PATH环境变量中

2. **macOS**：
   - 使用Homebrew安装：`brew install ffmpeg`

3. **Linux**：
   - 使用包管理器安装：`sudo apt install ffmpeg`（Ubuntu/Debian）

## 🚀 安装与运行

1. 📥 克隆或下载项目到本地
2. 📂 进入项目目录
3. 🔧 安装依赖：`npm install`
4. ▶️ 运行应用：`npm start`

## 📦 打包为可执行文件

### 打包为Windows可执行文件
1. 确保已安装所有依赖：`npm install`
2. 执行打包命令：`npm run package`
3. 🎉 打包完成后，可执行文件将生成在 `dist` 目录中

### 打包为macOS应用
1. 确保已安装所有依赖：`npm install`
2. 执行打包命令：`npm run build`
3. 🎉 打包完成后，应用将生成在 `dist` 目录中

### 打包为跨平台安装包
1. 执行构建命令：`npm run build`
2. 🎉 构建完成后，安装包将生成在 `dist` 目录中，包括Windows和macOS版本

## 📖 使用方法

1. 🚀 启动应用程序，它会自动扫描同级目录及其子目录中的电影
2. ▶️ 点击电影卡片可直接调用系统默认播放器播放电影
3. 👀 鼠标悬浮在电影卡片上可查看电影详细信息
4. 📁 点击左侧文件夹列表可导航到不同的文件夹
5. 📄 使用分页控件可浏览更多电影，支持自定义每页显示数量
